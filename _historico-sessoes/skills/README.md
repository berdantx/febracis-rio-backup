# Skills do projeto Febracis Rio

As 5 skills usadas neste projeto, em 2 grupos:

## 1. Customizadas (estao NESTA pasta — copiar para instalar)

| Skill | O que faz |
|---|---|
| `clone-redesign` | Clona paginas web existentes, baixa assets e reconstroi com design system proprio |
| `landing-cis-presencial` | Design system oficial do Metodo CIS (navy #0A1430 + amarelo #FFB901, Poppins/Inter, 17 folds) — nasceu deste projeto |
| `mentoria-reuniao` | Processa transcricoes/observacoes de reunioes de mentoria — fichas, planos, notas de sessao |

**Instalar em outro computador:** copiar cada pasta para `C:\Users\<usuario>\.claude\skills\`
(criar a pasta `skills` se nao existir). Na proxima sessao do Claude Code elas ja aparecem.

## 2. Plugins oficiais (NAO se copiam — se instalam pelo Claude Code)

| Skill | Como instalar |
|---|---|
| `frontend-design` | `/plugin` no Claude Code → marketplace `claude-plugins-official` → instalar `frontend-design` |
| `superpowers` (brainstorming etc.) | `/plugin` → marketplace `claude-plugins-official` → instalar `superpowers` |
